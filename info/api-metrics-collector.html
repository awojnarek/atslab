WORK IN PROGRESS: API Metrics Collector is a project born out of the following principles:
+ Remove the idea of "collect it messy and throw it over the ETL fence"
+ Three P's: Powerful, Performant and Pluggable
+ A single codebase that provides native binaries for (x86/Power/ARM):
- Linux
- AIX
- Windows
- MacOS
- OpenBSD
- Solaris
+ Any REST API. From zero to Galileo/Splunk/Whatever charts in under an hour.
+ Innovative features (gap fillers):
- "Near real-time" collection
- ASCII historical graphs on CLI
- Immediate and instantaneous Galileo support
- Any REST API is now able to be a Galileo integration
