Spectrum Protect Metrics Collector is a project born out of the following principles:
+ Spectrum Protect is (one of) our core technologies and deserves a robust agent.
+ Three P's: Powerful, Performant and Pluggable
+ A single codebase that provides native binaries for (x86/Power/ARM):
- Linux
- AIX
- Windows
- MacOS
- OpenBSD
- Solaris
+ Reliance on OC
+ No reliance on outside binaries. The only current outside libraries used are:
- viper (provide configurations)
- cobra (provide command-line flags / organization)
+ Innovative features (gap fillers):
- "Near real-time" collection
- Compression percentage by day/month/year
- Dedupe percentage by day/month/year
- Total raw nightly data ingestion
- SP Administrative metrics: reclaimation times, replications etc.
- Full Actlog analysis
